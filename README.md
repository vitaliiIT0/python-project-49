## Описание

Brain Games — набор из пяти консольных игр, развивающих логику и математические навыки.

## Минимальные требования

- Python 3.12 и выше

## Установка и запуск

1. Установите зависимости:
   ```bash
   make install
   uv install

2. Запуск игры

brain-even
brain-calc
brain-gcd
brain-progression
brain-prime
